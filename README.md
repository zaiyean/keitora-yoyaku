# 軽トラ予約システム　セットアップ手順

## 構成

| 役割 | 技術 |
|------|------|
| 画面（フロント） | HTML/CSS/JS（GitHub Pages で公開） |
| データ処理（バック） | Google Apps Script（GAS） |
| データ保存 | Google スプレッドシート |

---

## ① スプレッドシートの準備

1. Google スプレッドシートを新規作成する
2. 「設定」シートを作成し、1行目にヘッダーを入力する：

| A列 | B列 | C列 | D列 | E列 | F列 |
|-----|-----|-----|-----|-----|-----|
| 自治会ID | 自治会名 | 管理者パスワード | 質問項目 | 用途選択肢 | 通知メールアドレス |

3. 2行目以降に自治会の情報を入力する（例）：

| higashi | 東発区会 | 1234 | 氏名,組,電話番号,用途 | 引越し・荷物運搬,農作業・資材運搬,ゴミ・廃材処理,その他 | admin@example.com |

**ポイント：**
- 自治会IDは英字（例：higashi、nishi）
- パスワードは数字4桁
- 質問項目・用途選択肢はカンマ区切り（スペース不要）
- 予約シート（`higashi_予約` など）はGASが自動で作成します

4. スプレッドシートのURLから**スプレッドシートID**をメモする
   - 例：`https://docs.google.com/spreadsheets/d/`**【ここがID】**`/edit`

---

## ② GASのセットアップとデプロイ

1. スプレッドシートのメニュー → 「拡張機能」→「Apps Script」を開く
2. `Code.gs` の内容を貼り付ける
3. 1行目の `SPREADSHEET_ID` を自分のIDに書き換える：
   ```javascript
   var SPREADSHEET_ID = '貼り付けたID';
   ```
4. 「デプロイ」→「新しいデプロイ」をクリック
5. 設定：
   - 種類：**ウェブアプリ**
   - 次のユーザーとして実行：**自分**
   - アクセスできるユーザー：**全員**
6. 「デプロイ」ボタンを押して、表示された**デプロイURL**をコピーする

---

## ③ HTMLファイルへのURL設定

以下の4ファイルを開き、それぞれ冒頭の `GAS_URL` を書き換える：

- `index.html`
- `yoyaku.html`
- `admin.html`
- `admin_add.html`

```javascript
const GAS_URL = 'https://script.google.com/macros/s/【ここにデプロイURL】/exec';
```

---

## ④ GitHub Pages で公開

1. GitHub でリポジトリを作成（**Privateを推奨**）
2. ファイルをpushする
3. Settings → Pages → Branch: main → Save
4. 発行されたURLを自治会に共有する

---

## 使い方

### 一般会員（予約申請）
```
index.html → 自治会を選択 → 「一般会員」を選択 → yoyaku.html
```

### 役員（管理画面）
```
index.html → 自治会を選択 → 「役員・管理者」を選択 → パスワード入力 → admin.html
```

---

## 新しい自治会を追加するとき

1. スプレッドシートの「設定」シートに1行追加するだけ（コード変更不要）
2. 最初の予約が入ったとき、GASが自動でシートを作成します

---

## 質問項目・用途を変更するとき

1. スプレッドシートの「設定」シートの該当行を編集するだけ（コード変更不要）
